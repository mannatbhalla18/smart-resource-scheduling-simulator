# Smart Resource Scheduling Simulator

This project simulates and compares common CPU scheduling algorithms.

## Algorithms Implemented
- First Come First Serve (FCFS)
- Shortest Job First (SJF)
- Round Robin (RR)

## Features
- Random process generation
- Performance comparison between algorithms
- Metrics calculated:
  - Average Waiting Time
  - Average Turnaround Time
  - CPU Utilization
- Visualization dashboard with Gantt charts and comparison graphs

## Technologies Used
- Python
- Matplotlib
- NumPy
